# 学生成长档案画像系统

## 一、页面功能说明

# 1. 教师端首页

## 页面目标

让教师快速完成日常评价、审核、成绩导入和学生画像查看。

## 页面内容

1. 今日课程
2. 快速评价入口
3. 待审核学生自评
4. 待审核荣誉记录
5. 最近评价记录
6. 任教班级列表
7. 成绩导入入口
8. 学生画像入口

## 核心操作

- 点击“快速评价”，进入课堂动态评价页面。
- 点击“待审核”，进入审核中心。
- 点击“成绩导入”，进入成绩管理页面。
- 点击学生姓名，进入学生画像页面。

---

# 2. 课堂动态评价页面

## 页面目标

帮助教师快速记录一个或多个学生的课堂表现。

## 页面内容

### 第一步：选择学生

支持：

1. 选择班级
2. 单选学生
3. 多选学生
4. 按小组选择
5. 全班选择

### 第二步：选择评价入口

可选：

1. 科学
2. 英语
3. 数学
4. 语文
5. 礼仪
6. 运动
7. 艺术
8. 劳动

### 第三步：选择评价维度

可选：

1. 学习态度
2. 课堂常规
3. 学业水平
4. 竞赛活动

### 第四步：选择分值

可选：

- +1
- +2
- +3
- +5
- +10
- -1
- -2
- -3
- -5

### 第五步：选择原因卡片

根据评价维度展示正向或负面原因卡片。

例如选择“学习态度”后，展示：

- 正向原因
- 负面原因

### 第六步：补充说明

教师可填写文字说明。

### 第七步：提交评价

提交后，评价直接进入学生成长档案。

---

# 3. 学生自评提交页面

## 页面目标

让学生主动提交自己的课堂表现或成长表现。

## 页面内容

1. 选择评价入口
2. 选择评价维度
3. 选择分值
4. 选择原因卡片
5. 填写表现说明
6. 可上传证明图片
7. 提交审核

## 提交后状态

1. 待审核
2. 已通过
3. 已退回
4. 未通过

## 业务规则

学生提交后不直接计入画像，必须由教师审核通过后才生效。

---

# 4. 审核中心页面

## 页面目标

教师审核学生提交的自评记录和荣誉记录。

## 页面内容

### 筛选条件

1. 班级
2. 学生
3. 提交类型
4. 提交时间
5. 审核状态

### 审核列表

显示：

1. 学生姓名
2. 提交类型
3. 评价入口
4. 评价维度
5. 分值
6. 原因
7. 说明
8. 提交时间
9. 当前状态

### 审核操作

1. 通过
2. 退回修改
3. 不通过

## 审核通过后

记录进入学生成长档案，并参与画像计算。

---

# 5. 学科成绩管理页面

## 页面目标

导入并管理学生单元测试、期中考试、期末考试成绩。

## 页面内容

### 成绩导入区

1. 选择年级
2. 选择班级
3. 选择学科
4. 选择考试类型
5. 填写考试名称
6. 下载 Excel 模板
7. 上传 Excel
8. 系统校验
9. 确认导入

### 成绩列表区

显示：

1. 学生姓名
2. 学号
3. 学科
4. 考试名称
5. 考试类型
6. 满分
7. 得分
8. 考试时间

## 导入校验

系统校验：

1. 学生是否存在
2. 学号姓名是否匹配
3. 得分是否超过满分
4. 是否重复导入
5. 必填项是否为空

---

# 6. 日常竞赛和荣誉管理页面

4. 所属五育分类
5. 级别
6. 奖项

### 荣誉列表

1. 学生姓名
2. 活动名称
3. 活动类型
4. 五育分类
5. 级别
6. 奖项

8. 审核状态

## 业务规则

教师录入可直接生效。

学生提交荣誉需要教师审核后生效。

1. 教师首页
2. 课堂动态评价页
3. 批量评价确认页
4. 审核中心页
5. 成绩导入页
6. 荣誉录入页
7. 学生首页
8. 学生提交表现页
9. 学生提交荣誉页
10. 学生画像首页
11. 智育详情页
12. 班级分析页

这 12 个页面可以覆盖系统第一阶段核心闭环：

```text
评价录入 → 审核 → 成绩导入 → 荣誉录入 → 学生画像 → 班级分析
```
